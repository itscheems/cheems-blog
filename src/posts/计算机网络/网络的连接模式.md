# 摘要
excerpt: <p>桥接、NAT、仅主机，都分别是什么意思？</p>
---

## 桥接

指的是其他计算机/虚拟机，与当前主机处在同一个网段下，占用本网段的 ip，**并且这些计算机/虚拟机能与主机/其他计算机互相通信**（因为都在同一网段下）。此时主机和虚拟机之间是在同一层级的网络中，相当于直接用集线器或交换机直接连起来的一个局域网

## NAT

NAT 所构建的虚拟机网段，是主机网络下的二级网络，外部并不能直接通过 ip 来访问到虚拟机网络。NAT 用于多台计算机共用一个 ip 地址访问互联网，对外输出的是主机网卡的 ip 地址。（实际上可能会经过多重NAT转换）

一般虚拟机集群都是采用 NAT 来确定 ip，因为不占用主机网段的 DHCP 池资源，不容易造成 ip 冲突

## 仅主机

指的是虚拟机仅仅能与主机进行通信，但无法与外网之间互相通信
